Produire en local

La production tourne en local (voix neurale, rendu 1080p24, musique synthétisée, mixage, H.264) — aucune IA, aucun cloud.
⬇ Télécharger le MP4
Serveur local non détecté. Lancez-le dans un terminal :
cd video-debrief/pipeline python server.py # Windows python3 server.py # macOS / Linux
…puis revenez ici. Ou en ligne de commande directe :
python pipeline/build.py mon-projet.json # Windows python3 pipeline/build.py mon-projet.json # macOS / Linux

Cloud GitHub

Production sur GitHub Actions (~12 min) : le projet est poussé dans video-debrief/requests/, le workflow produit le MP4 et le publie dans la release videos. Token : fine-grained PAT du repo avec permissions Contents (read/write) et Actions (read).
